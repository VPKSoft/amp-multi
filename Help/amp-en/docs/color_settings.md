# Color settings

The color settings allows to specify colors for specified UI images via the color settings dialog.

The colors can be synchronized to use a single color if one wants a more peaceful UI eXperience. Just remember to set toggle the *Synchronize colors* checkbox. Also disabled colors such as the *Disabled button image color* might be useful to set in de-synchronized mode.
